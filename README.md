# mt8000_lib

Repository of extracted Anritsu MT8000A SCPI-like remote commands and helper scripts.

Contents
- `MT8000A_SCPI_Commands_Report.md` — comprehensive command reference extracted from the source document
- `mt8000a_commands.py` — helper script(s) for parsing/using commands
- `document.md` — original source notes

Quickstart
1. Clone: `git clone https://github.com/d1658999/mt8000_lib.git`
2. (Optional) Create and activate a virtual environment:

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

3. Install dependencies (if any): `pip install -r requirements.txt` (file may not exist)
